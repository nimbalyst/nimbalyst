# Welcome to Nimbalyst

Welcome to **Nimbalyst**, a powerful markdown editor built on Meta's Lexical framework. This guide will help you get started with the key features.

## Getting Started

### Opening Files
- Use **File > Open** or **Cmd+O** to open markdown files
- Drag and drop files into the editor
- Use the project sidebar to browse and open files from your workspace

### Creating Documents
- **File > New** or **Cmd+N** creates a new untitled document
- Start typing to begin editing
- Save with **File > Save** or **Cmd+S**

## Key Features

### Rich Text Editing
Nimbalyst supports rich markdown formatting:
- **Bold** and *italic* text
- Headers at multiple levels
- Lists (ordered and unordered)
- Code blocks with syntax highlighting
- Tables with full editing support
- Block quotes and horizontal rules

### AI-Powered Assistance
Press **Cmd+Shift+A** to open the AI Chat panel:
- Get help with writing and editing
- Generate content based on your document
- Ask questions about your work
- Multiple AI provider support (Claude, OpenAI, LM Studio)

### Project Management
- Open folders as workspaces to access all your files
- File tree navigation with drag-and-drop support
- Recent files for quick access
- Multiple windows for different projects

### Tables
Create and edit tables directly in your documents:
- Click and drag to resize columns
- Add/remove rows and columns
- Full keyboard navigation
- Export to various formats

### Code Blocks
```javascript
// Syntax highlighting for many languages
function example() {
  return "Hello, Nimbalyst!";
}
```

### Document History
Every document maintains its edit history:
- Undo/redo with **Cmd+Z** / **Cmd+Shift+Z**
- Full session history preserved
- Navigate through changes

## Themes

Nimbalyst supports multiple themes:
- **Light** - Clean, bright theme
- **Dark** - Warm dark theme
- **Crystal Dark** - Premium dark theme
- **Auto** - Follows system preference

Change themes via **Window > Theme** in the menu.
